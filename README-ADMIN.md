# ระบบผู้ดูแล Scratch AR Adventure

เปิดเกมด้วย `start-server.bat` หรือ Live Server แล้วเข้าเมนู “ผู้ดูแลระบบ”

- ชื่อผู้ใช้เริ่มต้น: `admin`
- รหัสผ่านเริ่มต้น: `admin1234`

ความสามารถ:
- ดูภาพรวมผู้เล่น คะแนน ดาว และจำนวนด่าน
- ค้นหา รีเซ็ต และลบผู้เล่น
- สร้าง แก้ไข ลบ และเผยแพร่ข้อมูลด่าน
- ดูและพิมพ์รายงานรายห้อง
- ส่งออกข้อมูลผู้เล่น JSON
- สำรองและกู้คืนข้อมูลทั้งหมด
- เปลี่ยนรหัสผ่านแอดมิน

หมายเหตุ: รุ่นนี้เก็บข้อมูลใน localStorage ของเบราว์เซอร์ เหมาะกับต้นแบบหรือใช้งานบนเครื่องเดียว รหัสผ่านฝั่งเบราว์เซอร์ไม่ใช่ความปลอดภัยระดับระบบออนไลน์จริง
