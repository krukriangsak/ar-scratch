# Camera Session + Lesson-linked Unlock

- อนุญาตกล้องและตรวจจับมือครั้งแรกจากหน้าแรก
- เมื่อเปลี่ยนหน้า ระบบจะเปิดกล้องและมือให้อัตโนมัติตามสถานะใน sessionStorage
- การนำทางแบบเปลี่ยนหน้าเว็บไม่สามารถส่ง MediaStream เดิมข้ามเอกสารได้ เบราว์เซอร์จึงเปิด stream ใหม่อัตโนมัติโดยไม่ต้องกดปุ่มซ้ำหลังเคยอนุญาตแล้ว
- ผู้เรียนต้องกด “เรียนจบและปลดล็อกด่าน” ในบทเรียน
- ด่านรองรับหลายบทเรียนที่ต้องเรียนก่อน และด่านก่อนหน้าที่ต้องผ่าน
- ผู้ดูแลแก้ความสัมพันธ์ได้ใน Level Builder
