# Scratch AR Adventure – Admin Studio

## เริ่มใช้งาน
1. แตกไฟล์ ZIP
2. เปิด `start-server.bat` บน Windows หรือ `start-server.sh` บน macOS/Linux
3. เปิดหน้าเกมตาม URL ที่แสดง
4. เข้า `admin.html`
5. ครั้งแรก ระบบจะให้ครูสร้างชื่อผู้ดูแลและรหัสผ่านใหม่เอง โดยไม่มีรหัสเริ่มต้นแสดงบนหน้าเว็บ

## ระบบที่เพิ่ม
- เนื้อหา Scratch ครอบคลุมพื้นฐาน หน้าจอโปรแกรม และหมวด Motion, Looks, Sound, Events, Control, Sensing, Operators, Variables & Lists, My Blocks และ Extensions
- ด่านแม่แบบ 18 ด่าน ตั้งแต่พื้นฐานจนถึง Final Project
- Scratch blocks ในด่านเป็นภาษาอังกฤษ
- ตัวจัดการบทเรียน เพิ่ม/แก้ไข/สำเนา/ลบ/เผยแพร่
- Level Builder ตั้งเวลา หัวใจ ดาว คำตอบ และ URL รูป/เสียง
- คลังสื่อสำหรับเก็บ URL รูป เสียง วิดีโอ และลิงก์
- ปรับ UI: สี ความโปร่งใส ความเบลอ ขนาดตัวอักษร และความโค้ง
- จัดการผู้เล่น รายงาน CSV สำรอง/กู้คืน JSON

## หมายเหตุด้านความปลอดภัย
เวอร์ชันนี้เก็บข้อมูลใน localStorage และใช้ SHA-256 สำหรับรหัสผ่านในเบราว์เซอร์ เหมาะกับต้นแบบหรือเครื่องเดียวในห้องเรียน สำหรับการใช้งานหลายเครื่องหรืออินเทอร์เน็ตจริงควรมี Backend และฐานข้อมูลกลาง
