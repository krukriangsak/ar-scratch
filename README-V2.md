# Scratch AR Adventure — Camera & Learning Edition

## วิธีเปิดที่ถูกต้อง
1. แตกไฟล์ ZIP
2. ดับเบิลคลิก `start-server.bat`
3. เปิด `http://localhost:8000` ใน Chrome หรือ Edge
4. กดอนุญาตกล้อง

ห้ามเปิด `index.html` ด้วยการดับเบิลคลิก เพราะเบราว์เซอร์อาจไม่อนุญาตกล้องและ JavaScript modules

## การแก้กล้อง
- ตรวจ secure context และแจ้งวิธีแก้
- ทดลองข้อกำหนดกล้อง 3 ระดับ จาก HD ไปถึงค่ามาตรฐาน
- แยกการเปิดกล้องออกจาก Hand Tracking: กล้องยังแสดงภาพได้แม้ MediaPipe โหลดไม่สำเร็จ
- ล็อกเวอร์ชัน MediaPipe เพื่อป้องกัน CDN เปลี่ยนไฟล์
- ข้อความผิดพลาดอธิบายกรณีปฏิเสธสิทธิ์ ไม่พบกล้อง หรือกล้องถูกโปรแกรมอื่นใช้งาน

## บทเรียนและเกม
- เนื้อหาไทยแบบเต็มเรื่องพื้นฐาน หน้าจอ หมวดบล็อก อัลกอริทึม Debugging และโครงงาน
- ชื่อบล็อกเป็นภาษาอังกฤษ
- สีบล็อกใกล้เคียง Scratch 3 ตามหมวด Motion, Looks, Sound, Events, Control, Sensing, Operators, Variables, My Blocks และ Extensions
- ปุ่มอ่านบทเรียนภาษาไทย
- ปุ่มอ่านชื่อบล็อกภาษาอังกฤษ และปุ่มเสียงบนแต่ละบล็อก
- ด่านเล่นจริงผ่าน `game.html` มีการเลือกบล็อก ตรวจคำตอบ หัวใจ เวลา ดาว และบันทึกความก้าวหน้า

## หมายเหตุ
ระบบนี้ใช้แนวคิดและชื่อหมวดเพื่อการเรียนรู้ โดยไม่ได้ฝังหรือลิงก์นักเรียนไปยังเว็บไซต์ Scratch จากบทเรียนเริ่มต้น
