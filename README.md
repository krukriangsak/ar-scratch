# Scratch AR Adventure

ระบบต้นแบบเกมการเรียนรู้ Scratch ป.4 พร้อม:
- เปิด/ปิดกล้อง
- ตรวจจับมือด้วย MediaPipe Hand Landmarker
- ใช้นิ้วชี้แทนเมาส์ และหนีบนิ้วเพื่อกด
- เสียงอ่านภาษาไทยด้วย Web Speech API
- โปรไฟล์และตัวละคร
- บันทึกข้อมูลด้วย localStorage

## วิธีเปิด
กล้องไม่ทำงานเมื่อเปิดไฟล์ด้วย `file://` จึงต้องเปิดผ่าน Live Server หรือ localhost

### วิธีที่ 1: VS Code
1. เปิดโฟลเดอร์นี้ใน VS Code
2. ติดตั้งส่วนขยาย Live Server
3. คลิกขวา `index.html` แล้วเลือก **Open with Live Server**

### วิธีที่ 2: Python
เปิด Terminal ในโฟลเดอร์แล้วรัน:

```bash
python -m http.server 5500
```

จากนั้นเปิด `http://localhost:5500`

## หมายเหตุ
ระบบตรวจจับมือต้องใช้อินเทอร์เน็ตครั้งแรกเพื่อโหลด MediaPipe และโมเดลจาก CDN

## การแก้ระบบตรวจจับมือ
เวอร์ชันนี้ใช้ MediaPipe Hands ผ่าน CDN Global Script แบบเดียวกับโค้ดตัวอย่างที่ทำงานได้
จึงต้องเชื่อมต่ออินเทอร์เน็ตขณะเปิดเกมครั้งแรก และต้องเปิดผ่าน Live Server หรือ localhost

## การแก้ไขภาพกล้องล่าสุด
- ภาพจากกล้องหน้าแสดงแบบกระจกตามธรรมชาติ
- โครงกระดูกมือและตัวชี้อยู่ฝั่งเดียวกับมือในภาพ
- แสดงภาพกล้องเต็มความชัด ไม่มีชั้นสีโปร่งแสงบังใบหน้า
