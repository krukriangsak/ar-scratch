# Scratch AR Adventure v3.3 — Admin Studio & Sprite Simulator

ระบบต้นแบบเกมการเรียนรู้ Scratch สำหรับนักเรียนชั้นประถมศึกษาปีที่ 4 พร้อมกล้อง ตรวจจับมือ เสียงอ่าน ระบบบทเรียน ด่าน คะแนน และผู้ดูแลระบบ

## จุดเด่นเวอร์ชัน 3.2

- ปรับหน้า Admin Studio ใหม่เป็น Sidebar ใช้งานง่ายและสมมาตร
- เพิ่มหน้าแก้ไข UI สำหรับสีหลัก สีเน้น ความทึบ ความโปร่งใส ความเบลอ ความโค้ง และขนาด UI
- มีธีมสำเร็จรูป Pastel, Ocean, Forest, Sunset และ Dark Galaxy
- การตั้งค่า UI จาก Admin เชื่อมกับหน้าผู้เล่นผ่าน localStorage
- หน้าผู้เล่นไม่มีเมนู Admin ซ้ำ เหลือเพียงปุ่มผู้ดูแลระบบมุมหน้าจอ
- เพิ่มคลังบล็อก Scratch 9 หมวด พร้อมสีและรูปทรงแยกประเภท
- รองรับรูปทรง Stack, Hat, Reporter, Boolean, C-Block, Cap และ Define
- เพิ่ม Stage Preview และตัวละครที่ทำตามคำสั่ง Motion, Looks, Variables และ Wait แบบพื้นฐาน
- แสดงค่า X, Y, Direction และ Score ระหว่างทดลองรัน
- ผู้ดูแลแก้ไขบทเรียน ด่าน คะแนน คำตอบ และเงื่อนไขปลดล็อกได้

## การเปิดใช้งาน

กล้องไม่ทำงานเมื่อเปิดด้วย `file://` ต้องเปิดผ่าน localhost

### Windows

ดับเบิลคลิก `start-server.bat` แล้วเปิด:

`http://localhost:8000`

### Python

```bash
python -m http.server 8000
```

จากนั้นเปิด `http://localhost:8000`

## อัปเดต v3.3
- ตัวชี้นิ้วมีจุดกดสีแดง ป้ายชื่อเป้าหมาย และกรอบไฮไลต์ปุ่มที่กำลังชี้
- หนีบนิ้วโป้งกับนิ้วชี้เพื่อกด
- ชูนิ้วชี้และนิ้วกลางพร้อมกัน แล้วขยับมือขึ้นหรือลงเพื่อเลื่อนหน้า
- บทเรียนแบ่งเป็นหมวดใหญ่: รู้จักโปรแกรม Scratch, หมวดหมู่คำสั่ง Scratch และประยุกต์สร้างชิ้นงาน
- แผนที่ด่านแบ่งตามโลกการเรียนรู้ และบอกบทเรียนที่ต้องเรียนก่อนปลดล็อก
- Admin แสดงบทเรียนและด่านเป็นรายชื่อ กดแก้ไขเพื่อเปิดแบบฟอร์มเฉพาะรายการ ลดความสับสนจากแบบฟอร์มยาวทั้งหมด
